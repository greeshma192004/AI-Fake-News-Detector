# AI Fake News Detector

Developed a machine learning-based fake news detection system using Python and NLP techniques.

## Technologies
- Python
- Pandas
- Scikit-Learn
- TF-IDF Vectorization
- Logistic Regression
- Streamlit

## Features
- Detects Fake/Real News
- Confidence Score Prediction
- Web Interface
- Text Preprocessing

## Accuracy
98.69%
